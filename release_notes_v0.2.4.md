# Gaussian Scene Workbench 0.2.4

## English

- Added a direct **Export PLY** action to the Output toolbar.
- The export streams the original trained `point_cloud.ply` without conversion, preserving spherical harmonics, opacity, scale, and rotation attributes.
- Added a dedicated download endpoint and a descriptive attachment filename.
- Renamed the asset entry to **Gaussian PLY model** and localized it in English, Chinese, and Japanese.

## 中文

- 在“输出”工具栏加入直接的 **导出 PLY** 功能。
- 导出时直接传输训练产生的原始 `point_cloud.ply`，不做重编码，完整保留球谐系数、透明度、缩放和旋转属性。
- 新增专用下载接口和清晰的附件文件名。
- 资产项改名为 **Gaussian PLY 模型**，并适配中、日、英三语。

## 日本語

- 「出力」ツールバーに **PLY 保存** を追加しました。
- 学習で生成された元の `point_cloud.ply` を変換せずに出力し、球面調和関数、透明度、スケール、回転属性を保持します。
- 専用ダウンロード API と分かりやすい保存ファイル名を追加しました。
- アセット名を **Gaussian PLY モデル** に変更し、日本語・中国語・英語に対応しました。
