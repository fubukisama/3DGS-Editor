# Gaussian Scene Workbench 0.2.0

Updated: 2026-07-10

This release renames 3DGS Editor to **Gaussian Scene Workbench**, reflecting its broader role as a research environment for Gaussian scene reconstruction, experimentation, analysis, editing, rendering, and export.

## Highlights

- Introduces the professional product name **Gaussian Scene Workbench** and Chinese name **高斯场景研究工作台**.
- Renames the Windows executable, setup tools, package, window title, interface branding, and release documentation.
- Updates the Windows package version to `0.2.0`.
- Rewrites the README in English, Chinese, and Japanese around the complete research workflow.
- Keeps `GS_EDITOR_*` environment variables, existing Conda environments, legacy runtime folders, and scene data formats compatible.
- Includes the Experiment Manager, checkpoint resume and cloning, parameter diffs, training curves, GPU-visible Gaussian selection, resizable panels, adaptive UI scaling, rendering, PSNR analysis, and export workflows from the 0.1.x series.

Package SHA256 is published as a separate `.sha256` Release asset.
